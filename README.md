# crypto-briefing-agent
An autonomous Solana briefing agent for the @OOBEonSol × @AceDataCloud bounty.  Every hour it: → Scrapes Solana news (AceData SERP API) → Summarises with GPT-4o-mini (AceData LLM) → Generates a visual card (AceData Image API) → Settles payments via x402 → Writes results to SAP on-chain ledger
